## Controls

<b>In 3D</b>
- Use <b>arrow keys</b> to move.
- <b>Shift</b> to go faster.
- <b>T</b> to change view.

<b>In 2D</b>
- Use <b>mouse</b> to move.
- <b>Left and Right arrows</b> to rotate.
